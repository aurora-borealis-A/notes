# 一、矩阵

## 1. 矩阵定义

eye(num_dim)：单位矩阵

## 2. 矩阵运算

det(A)：求矩阵的**行列式**。

inv(A)：求矩阵的**逆**。

rank(A)：求矩阵的**秩**。

[E, V] = eig(A)：求矩阵的**特征向量**和**特征值**。

- E：特征向量矩阵，每一列代表一个特征向量（可能是复数）。
- V：特征值矩阵，对角线元素为特征值，其余元素都为零。
