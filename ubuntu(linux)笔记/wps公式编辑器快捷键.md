`Ctrl + g` ：希腊字母

用法：

`Ctrl + g`后，再键入a则得到希腊字母α，键入A则得到大写的希腊字母



`Ctrl + k`：符号

`Ctrl + k` + `,` ：小于等于

`Ctrl + k` + `.` ：大于等于

`Ctrl + k` + `t` ：乘号

`Ctrl + k` + `o` ：无穷（∞）

`Ctrl + k` + `e` ：属于（∈）



`Ctrl + f`：（fraction）分数

`Ctrl + /`：斜杠分式



`Ctrl + l`（low）：下标

`Ctrl + h`（high）：上标

`Ctrl + j`：上下标



`Ctrl + i`（integrate）：积分符号



`Ctrl + 9` 或 `Ctrl + 0`：小括号

`Ctrl + [`：中括号

`Ctrl + {` 或 `Ctrl + shift + [`：大括号



`Ctrl +  space`：小空格

`Ctrl + shift + space`：大空格



`Alt + i` + `q`：从wps打开公式编辑器

`Alt + f` + `x`：保存并关闭公式编辑器
